Portfolio management

Portfolio selection

The user can select stocks from the FTSEMIB index and investigate their expected returns and risk by considering historical data over the selected time window prior to the start date. A desired number of Monte Carlo sampled portfolios is generated by the application and the user can visualize the efficient frontier and the optimal portfolios, as well as the returns of the selected portfolios over one year after the start date, evaluated using historical data.
